The sidebar renders one row per agent pane across all tmux sessions and windows. Each row is packed with live metadata pulled from the agent's hooks and from tmux itself.

## Filter bar

The top row of the sidebar filters the agent list by status. Each entry is an icon + live count; selecting one hides rows whose status doesn't match.

### Status filter

- `‚â°N` ‚Äî all panes
- `‚óèN` ‚Äî running
- `‚óêN` ‚Äî waiting
- `‚óãN` ‚Äî idle
- `‚úïN` ‚Äî error

Cycle with `h` / `l` (or `‚Üê` / `‚Üí`), or click an entry. `Tab` advances through the filter one step at a time.

### Repo filter

Below the status row, a repo selector (`‚ñæ`) narrows the list to a single repository. Open it with `r` or by clicking the arrow; `‚Äî` means no repo filter is active.

## Row elements

### Status icon

- `‚óè` running
- `‚óê` waiting
- `‚óã` idle
- `‚úï` error

### Permission badge

- `plan` ‚Äî plan mode
- `edit` ‚Äî auto-edit
- `auto` ‚Äî auto-accept
- `dontAsk` ‚Äî skip every permission prompt for the session
- `defer` ‚Äî wait for an explicit decision before applying
- `!` ‚Äî bypass permissions (dangerous)

### Session name

Defaults to the agent name. Can be changed through Claude Code's `/rename`.

### `+` marker

Indicates the pane sits in a git worktree (any worktree, not only sidebar-created ones). Sidebar-spawned worktrees additionally show a `√ó` button for one-keystroke teardown. See [Worktree](/tmux-agent-sidebar/features/worktree/) for details.

### Branch

Current Git branch for the pane's cwd. Claude updates dynamically through the `CwdChanged` hook.

### Elapsed time

Time since the last user prompt.

### Task progress

Synchronized from the agent's task list. Each task gets a glyph for its state; the trailing count is `completed/total`.

- `‚úî` ‚Äî completed
- `‚óº` ‚Äî in progress
- `‚óª` ‚Äî pending

```text
  ‚úî‚úî‚óº‚óª 2/4
```

### Subagent tree

Parent-child branches for spawned sub-agents. Middle entries use `‚îú`, the last uses `‚îî`. Each row shows the subagent kind followed by `#<id>`.

```text
‚îú Explore #a1b2
‚îú Plan    #d4e5
‚îî Bash    #dead
```

### Listening ports

Localhost ports the pane's process is listening on.

### Response arrow (`‚ñ∑`)

Preview of the latest agent response.

### Prompt text

Latest user prompt.

### Wait reason

Why the agent is waiting ‚Äî populated from Claude's `Notification`, `PermissionDenied`, and `TeammateIdle` hooks.
